**5-day trend classification rule.**

```
five_day_pct = (spot_today − spot_5d_ago) / spot_5d_ago × 100
if abs(five_day_pct) <= 0.5:
    label = "flat"
elif five_day_pct > 0:
    label = "up"
else:
    label = "down"
```

Render as `<label> (<signed_pct>%)` with one decimal.

**Distance from 50d SMA rule.**

```
dist_pct = (spot − sma_50) / sma_50 × 100
direction = "above" if dist_pct >= 0 else "below"
```

Render as `<signed_pct>% (<direction>)` with one decimal.
