## Relevant Context — Do Not Deviate Without Escalation

**Source: auth-process.md (canonical)**
BFF OIDC is the only sanctioned auth path. PKCE + signed state + HttpOnly cookie + server-side session. No JWT returned to the browser anywhere.

**Source: architecture-plan.md § Pattern 6**
Auth flows that hand a JWT to the browser violate Pattern 6. `entra_auth_routes.py` does this and must be retired.

**Source: ticket OTA-538 → Three current routes**
- `identity_routes.py` — BFF OIDC. KEEP.
- `auth_routes.py` — legacy local-password. RETIRE entirely OR restrict to `/register` only if Don confirms registration is still in scope. Confirm with Don in Phase 1.
- `entra_auth_routes.py` — legacy MSAL bridge. RETIRE entirely (no remaining use case per ticket).

**Source: ticket OTA-538 → Resolver merge**
`dependencies.py` has two near-identical resolvers (`get_current_user` and `get_session_user`). After legacy retirement, merge into one resolver.

**Source: ticket OTA-538 → skip_auth audit + Key Vault failure mode**
- `skip_auth=True` must be impossible in production. Add a startup assertion.
- Any required Key Vault secret missing on startup must fail loudly, not silently auto-generate-and-fall-back.

**Source: CLAUDE.md → House Style + async credential rule**
All async Azure SDK calls must use `azure.identity.aio` async variants in async handlers. The startup-assertion code touches this surface — use the async credential.

2. Update every call site found in Phase 1 to import the merged name.

### skip_auth hardening

In application startup (likely `main.py` or `app/core/lifespan.py`):

```python
import os
if settings.skip_auth and os.environ.get("ENVIRONMENT", "").lower() == "production":
    raise RuntimeError(
        "FATAL: skip_auth=True is not permitted in production. "
        "Disable skip_auth or set ENVIRONMENT to a non-production value."
    )
```

Place the assertion early — before any router include or DB connection. The app must fail to start, not start in a degraded state.

### Key Vault fail-loud

In the Key Vault secret-fetch helper, replace any silent-fallback path with a hard raise on missing-required-secret. Document which secrets are required (BFF session signing key, Foundry API key, Schwab cert credentials, Jira API token) vs. optional. Use the async credential (`azure.identity.aio`) in async startup paths per CLAUDE.md.
